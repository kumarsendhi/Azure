Lab Overview

In this lab, you will use Visual Studio Community 2017 to create a new Azure Resource Manager Template that deploys a virtual machine scale set (VMSS) that can dynamically scale up or down. Each instance of the scale set will automatically deploy a web application on provisioning by using a PowerShell Desire State Configuration (DSC) extension.

Objectives

·         Understand how to deploy a load balancer in an ARM template

·         Understand how to deploy a virtual machine scale set in an ARM template

·         Understand how to configure PowerShell DSC extensions in a scale set

·         Understand how to deploy auto-scale rules

2 hours and 25 minutes
