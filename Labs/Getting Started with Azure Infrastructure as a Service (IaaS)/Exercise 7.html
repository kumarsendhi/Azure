Exercise 7: Create the Load Balancer

In this exercise, you will create a new load balancer that be used to distributed HTTP/Web traffic to the two virtual machines.

Task 1: Create the Load Balancer

1.     Open the Azure portal. Click New in the upper left hand corner. In the search menu type Load Balancer. Click to select the first option.

 

2.     In the dialog that opens, click Create at the bottom.

 

3.     In the Create load balancer dialog, enter OPSLB for the name. Leave the Scheme set to Public.

 

4.     Click Choose a public IP address then click Create new.

 

5.     Enter the name opsstaticip for the public IP name and set the Assignment set to Static. Then click OK.

 

6.     Ensure the correct subscription is selected.

 

7.     Change the Resource group to the OpsVMRmRG resource group.

 

8.     Ensure the Location is set to the same location used throughout this lab, then click Create.
