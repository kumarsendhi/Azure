Exercise 3: Update the Virtual Network

In this exercise, you will learn how to extend an Azure Virtual Network by adding another subnet to it.

Task 1: Add the Data subnet

1.     Within the OpsVNETRmRG resource group, click the virtual network OpsTrainingVNET.

2.     Click the Subnets tile on the virtual network configuration blade.

 

3.     Click Add

 

4.     Specify the following configuration on the new subnet.

a.     Name: Data

b.     Address range (CIDR block): 10.0.1.0/24

c.      Network security group: None

d.     Route table: None

5.     Click OK.
