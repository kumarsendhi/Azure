Lab Overview

In this lab, you learn about the deployment of an Azure Recovery Services Vault, followed by the implementation of the different Azure Backup Solutions (Azure Backup Agent-based, Azure Backup Server and Azure IaaS VM) available in Azure Resource Manager today. In the last exercise, you deploy the preview integration of Azure ARM backup with OMS.

Learning Objectives

·         Configure and deploy Azure Recovery Services Vault

·         Implement Azure Backup Agent-based backup solution

·         Implementing Azure Backup Server (MABS) based backup solution

·         Implementing Azure VM IaaS Backup solution

·         Monitoring Azure Resource Manager backup using OMS

Task 1: Deploying the Environment

In this first task, you will deploy the environment used throughout the lab.

1.    Download the student files to this exercise to your local computer and unzip them into a local folder.

 

2.    Launch the Azure Management portal by navigating to https://portal.azure.com and logging in with your credentials.

 

3.    Click +New and type in template and press enter. Click Create, and then click Build your own template in the editor.

 

4.    Click Load file, and browse to the downloaded student files and select the azuredeploy.json file. Click Save.

 

5.    Specify OpsBackupRG as the resource group name, choose the closest region to you for location, and check the box I agree to the terms and conditions stated above, and click Purchase.

 

 

Note:  This template deploys 4 virtual machines that will be used throughout this lab.

AgentBackupVM1

IISBackupVM1

MABSServerVM1

MABSADDSVM1   

 

This deployment may take 20-30 minutes.
