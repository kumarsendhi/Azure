Exercise 6: Delete Resources in your Azure Subscription

 

Overview

In this exercise, you will learn how to Azure ServiceBus with .NET Core.

Time Estimate

·       5 minutes

 

Task 1: Delete Resources in your Subscription

In this task, you will delete the resource groups for the resources you created in this lab. This will delete all the artifacts created in Azure for this lab.

1.     In the Azure Management portal, click Resource Groups.

 

2.     Click the OPSLABRG resource group.

 

3.     Click the Delete button to delete the resource group. When prompted type in the name of the resource group to confirm.

 

Do the same for the resource group Opsgility-sb-RG
