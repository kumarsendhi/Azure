Exercise 5: Copying blobs in Azure

In this exercise, you will extend the console app to learn how to copy blob files within the same Azure Storage container.

Time Estimate

·       10 Minutes

 

  1. Now let’s copy a blob to another blob with a different name.  To do this, you need to get a reference to the source blob and a reference to the destination blob. Then you call the StartCopy method on the destination blob and pass it the source blob.

 

We’re just going to do this in the same container. Add a method called CopyBlob; this is the code that goes in that method.

public void CopyBlob()

{

    //copy from image1name

    cloudBlockBlob = cloudBlobContainer.GetBlockBlobReference(image1Name);

            

    //get a reference to the destination blob.

    CloudBlockBlob destBlob = cloudBlobContainer.GetBlockBlobReference("copyof_" + image1Name);

 

    //start the copy from the source blob (cloudBlockBlob) to the destination blob.

    destBlob.StartCopy(cloudBlockBlob);

 

    GetListOfBlobs();

}

  1. Now go back up to BasicBlobOps and add a call to CopyBlob and then GetListOfBlobs before the console writes at the bottom.

public void BasicBlobOps()

{

    //set up the objects

    SetUpObjects();

    UploadBlobs();

    GetListOfBlobs();

 

    DownloadBlobs();

 

    CopyBlob();

    GetListOfBlobs();

 

    Console.WriteLine("Press any key to exit.");

    Console.ReadLine();

}
