Exercise 6: Free Resources in your Subscription

In this exercise, you will delete the resource groups for the resources you created in this lab. This will delete all the artifacts created in Azure for this lab.

Time Estimate

5 minutes

 

1.     In the Azure Management portal, click Resource Groups.

 

2.     Open the OpsVMRmRG resource group.

 

3.     Click the Delete button to delete the resource group. When prompted type in the name of the resource group to confirm.

 

4.     Repeat the process for the following resource groups (in order):

a.     OpsVNETRemoteRG

b.     OpsVNETRmRG

c.      OPSLABRG
